# subject_4-1-2_3_submission

## 課題4-1-2
提出用課題  
課題3：グラフ作成と画像保存

スコアデータのCSVを読み込み、
円グラフ・棒グラフ・ヒストグラムを作成し、
画像データとして保存するシステムです。

【実行方法】
1. Python 3.x がインストールされていることを確認
2. 必要なライブラリをインストール
   pip install pandas matplotlib
3. 以下のコマンドで実行
   python scoregraph.py

※ 実行すると、以下の画像ファイルが生成されます
- scores_by_dept.png
- participants_pie.png
- score_histogram.png
